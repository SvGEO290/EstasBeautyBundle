# ESTAS Beauty - Luxury Scar Care

A modern, responsive website for ESTAS Beauty's luxury scar care products.

## Features

- Responsive design
- Interactive product showcase
- Customer reviews carousel
- Press coverage section
- Analytics integration (Google Analytics 4 & Hotjar)

## Setup

1. Clone the repository:
```bash
git clone https://github.com/YOUR-USERNAME/estas-beauty.git
```

2. Open `index.html` in your browser to view the site locally.

## Deployment

This site is hosted on GitHub Pages. The main branch is automatically deployed to:
`https://YOUR-USERNAME.github.io/estas-beauty/`

## Analytics

The site includes:
- Google Analytics 4 for visitor tracking
- Hotjar for heatmaps and user behavior analysis

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Analytics 4
- Hotjar
